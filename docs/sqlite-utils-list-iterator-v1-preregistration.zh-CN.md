# sqlite-utils 自然 handoff 消融实验预注册

## 1. 实验目的

本实验检验六字段 handoff 对自然代码接续任务的作用。任务来自真实开源项目的功能需求，
不是为某个 handoff 字段定制的人工业务规则。后续 Agent 需要在固定 TASK-001 代码状态上，
完成公开批量写入接口对 positional row iterator 的支持。

固定 handoff schema：

1. `goal`
2. `changed_files`
3. `verification`
4. `unfinished_items`
5. `risks`
6. `next_steps`

实验组为完整组、六个 leave-one-field-out 组和无 handoff 负对照组，共 8 组。

## 2. 预注册假设

### H1：unfinished_items 影响最终正确性

与 A_full 相比，E_no_unfinished_items 的预期表现为：

- 首轮隐藏测试通过率更低；
- 最终 10/10 隐藏测试达成率更低；
- 返工次数和交互轮数更高。

理由：该字段描述前序阶段与后续阶段的真实完成边界。删除后，Agent 需要从代码中重新判断
哪些内部能力已完成、哪些公开入口仍未接通。

### H2：verification 影响执行效率

与 A_full 相比，D_no_verification 的预期表现为：

- Token 总量更高；
- 测试运行次数更多；
- 编辑次数或无效编辑轮次更多；
- 首轮成功率更低或返工次数更高。

理由：验证证据提供了可信的前序基线。删除后，Agent 可能重复验证前序工作或扩大排查范围。

### 探索性分析

B、C、F、G 与 H 用于探索目标、变更定位、风险和下一步建议对正确性、效率及行为路径的影响。
这些组不用于事后修改 H1/H2。

## 3. 固定条件

- Agent：Codex，同一模型版本和推理强度。
- 每个 run 使用新的 Codex 对话。
- 每个 run 从同一 TASK-001 tag 复制独立 workspace。
- 初始 prompt 除 arm 可见字段外完全相同。
- 最多 3 次 attempt；修复必须在同一对话中完成。
- 不允许访问 Git 历史、远端仓库、其他 run、HALF 协作仓库或私有评测目录。
- 完整 handoff 和隐藏测试只存在于 HALF 私有实验根目录。

## 4. 主要指标

### 正确性

- 首轮准确率：首轮隐藏测试通过数 / 10。
- 最终准确率：最后一轮隐藏测试通过数 / 10。
- 首轮成功：公开测试全通过且隐藏测试 10/10。
- 最终成功：最多 3 轮内公开测试全通过且隐藏测试 10/10。
- 公开测试通过率：公开 passed / total（skipped 单独记录）。

### 成本

- `total_tokens`：同一 run 全部 attempt 的真实 Codex Token 合计。
- 相对 Token：该组 Token / 同批次 A_full Token。
- 返工次数：`attempts - 1`。
- 交互轮数：提交 attempt 的总数。
- 人工干预次数与分钟数；仅业务性提示计入，掉线和服务故障记为 infra retry。

### 行为事件

从完整 tool trace 统计：

- 读文件次数与唯一文件数；
- 搜索次数与重复搜索次数；
- 测试运行次数；
- 编辑次数；
- 接口猜测次数；
- 无效编辑轮次；
- 回滚次数；
- 变更文件数。

## 5. 固定质量门

每个 attempt 必须提交完整 trace。HALF 扫描：

1. Agent 可见 prompt；
2. Agent 最终输出；
3. 全部工具调用 trace。

任何被删除字段的原文签名或 canary 命中都将该 run 标记为泄漏，不进入效果统计。
结果表必须报告“泄漏审计状态、trace 是否完整、删字段命中数”。

## 6. 重复与分析

- 先执行 A、D、E、H 各 1 次作为管线 pilot。
- pilot 必须满足：A 可解、H 非必然可解、泄漏审计通过、隐藏评测可区分未完成态与完成态。
- pilot 通过后，每个 A-H arm 至少运行 3 次。
- 比较中报告中位数和每次原始值，不只报告均值。
- 单次结果只作为探索性证据，不作为字段因果结论。

## 7. 解释边界

本任务不是“删字段必挂对应测试”的定向设计。某字段删除后无性能下降，可由两种解释产生：

1. 信息可从代码恢复；
2. 本任务对该信息需求较低。

因此结论限定为该任务分布上的经验结果，后续仍需在其他自然任务上复现。
